# ゴールドジム MAP（東京・神奈川・千葉・埼玉）

東京・神奈川・千葉・埼玉のゴールドジム全54店舗を地図上に表示する静的Webアプリ。

## 公開URL

https://denemon.github.io/goldsgym-kanto/

## 機能

- **店舗マップ（`index.html`）** … 全54店舗を地図にマーカー表示。都県タブ（すべて/東京都/神奈川県/千葉県/埼玉県）・店名/エリア/駅名での検索・「24時間営業のみ」フィルタ。ピンや一覧から最寄り駅・住所・Googleマップ導線を表示。
- **15分圏マップ（`zone.html`）** … 選んだ店舗から徒歩/自転車/車で15分圏内を円ゾーンで描画。「全店まとめて表示」でカバー範囲を俯瞰。
  - 徒歩 半径1.2km（80m/分） / 自転車 半径3.5km（233m/分） / 車 半径5.0km（333m/分）
  - 道路を無視した直線距離ベースの概算（円バッファ方式）。

## 構成

| ファイル | 役割 |
|----------|------|
| `index.html` / `app.js` | 店舗マップ |
| `zone.html` / `zone.js` | 15分圏マップ |
| `styles.css` | 黄色グラデーションのデザイン（両ページ共通） |
| `gyms.json` | 店舗データ（店名・最寄り駅・住所・座標）。**ここだけ直せばデータ更新できる** |

地図の背景は [Leaflet](https://leafletjs.com/) + OpenStreetMap（無料・APIキー不要）。
店舗位置などの「事前に用意した情報」は `gyms.json` に持たせ、その上に描画する構成です。

## ローカルで動かす

`fetch` でJSONを読むため、ファイルを直接開くのではなく簡易サーバー経由で開きます。

```bash
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## データ更新

`gyms.json` の各店舗オブジェクトを編集するだけ。座標(`lat`/`lng`)は国土地理院ジオコーディング
（住所→座標）で取得した値で、番地レベルの中心点のため建物と数十m程度ずれる場合があります。
ピンポイントに合わせたい場合は `lat`/`lng` を個別に書き換えてください。
